# AI 协作约束

本文件约束 AI 助手在 OpsMind 仓库中的工作方式。

## 职责划分

- 用户重点负责后端、数据库、基础设施、部署和 Git 操作。
- AI 助手重点负责前端，可以直接创建、修改、删除和验证 `frontend/` 目录内的文件。
- 对于后端、数据库、基础设施和部署工作，AI 助手只提供操作步骤、命令、代码示例、检查方法和问题分析。
- 未经用户明确要求，AI 助手不得代替用户修改 `backend/`、`deploy/` 或其他非前端业务文件，不得安装依赖、启动服务、运行迁移或执行部署。
- AI 助手不得代替用户执行任何 Git 操作，包括 `git init`、`git add`、`git commit`、`git pull`、`git push`、分支切换、合并、变基和标签操作。
- 未经用户明确要求，AI 助手不得执行会改变本机、仓库或远程环境状态的命令。

## 前端授权

- AI 助手可以直接完成 `frontend/` 下的工程初始化、页面开发、组件开发、样式调整、接口适配和前端测试。
- AI 助手可以在前端开发需要时修改根目录中与前端直接相关的配置文件，但应提前说明修改内容。
- AI 助手可以运行只影响前端的格式化、类型检查、测试和构建命令。
- AI 助手可以直接启动、停止和重启前端开发服务，无需再次获得用户授权。
- 如需安装前端依赖，AI 助手应先向用户说明并获得明确授权。
- AI 助手不得将前端授权扩大到后端、数据库、部署或 Git 操作。

## 本地运行默认授权

- OpsMind 本地开发环境视为安全环境。AI 助手可以直接执行常规运行操作，无需逐次询问用户是否允许。
- 默认授权范围包括：启动、停止和重启本地开发服务；运行本地 Docker Compose 服务；查看日志和端口；执行健康检查；运行格式化、类型检查、测试和构建命令。
- 为完成用户当前明确要求的本地验证或故障排查，AI 助手可以直接执行必要的只读检查和非破坏性运行命令。
- 默认授权不包括：安装或升级依赖、运行数据库迁移、删除或重置数据、修改生产或远程环境、执行部署，以及任何 Git 操作。
- 对于默认授权范围外的操作，仍需用户在当前对话中明确要求或授权。

## 后端协助方式

- 根据当前任务给出按顺序排列的操作步骤。
- 提供可由用户手动执行的终端命令。
- 提供需要新增或修改的文件路径与代码内容。
- 解释每一步的目的、预期结果和验证方式。
- 在用户提供报错、日志或代码后分析原因并给出修复步骤。
- 用户明确要求检查时，可以执行只读操作，例如读取文件、查看目录和检查 Git 状态。

## 命令讲解规则

- 当要求用户执行此前未讲解过的命令时，必须解释命令语法、参数含义、执行目的和预期结果。
- 对于已经向用户讲解过的命令，无需重复解释；仅在参数、用途或执行环境发生变化时补充说明。
- 命令较长或包含多个参数时，应按组成部分拆分讲解，确保用户能够理解后再手动执行。

## 代码讲解规则

- 当指导用户编写后端代码时，首次出现的第三方库、Python 标准库模块和库函数必须讲解。
- 讲解内容至少包括：库或函数来自哪里、解决什么问题、当前代码为什么需要它，以及关键参数和返回值。
- 对于已经讲解过的库和函数，无需机械重复；仅在使用方式、参数或作用发生变化时补充说明。
- 示例代码应按当前步骤控制规模，避免一次引入过多尚未讲解的概念。

## 输出要求

针对后端、数据库和部署任务，AI 助手应优先按以下结构回答：

1. 当前目标
2. 需要修改的文件
3. 用户需要执行的步骤
4. 验证命令与预期结果
5. Git 提交建议

当任务较小时，可以适当缩短，但必须确保用户能够独立完成操作。针对前端任务，AI 助手可以直接实施并报告修改内容和验证结果。

## 下一步指引规则

当用户询问“下一步干啥”“接下来做什么”或表达同类含义时，AI 助手必须根据 `docs/module-plan.md` 返回用户需要亲自完成的任务。

- 按模块顺序推进，例如 `M0：项目骨架`、`M1：用户与权限`。
- 默认一次只安排一项当前最优先的任务，不要一次性要求用户完成整个模块。
- 明确标注模块、任务序号和进度状态。
- 已完成的任务标注为 `已完成`。
- 当前需要用户处理的任务标注为 `当前任务`。
- 尚未开始的任务标注为 `待处理`，仅在有助于理解上下文时简要列出。
- 如果无法确认上一项是否完成，先要求用户提供结果或执行验证命令。
- 前端任务由 AI 助手直接实施，不列入用户待办；必要时说明 AI 助手将在什么条件满足后开始处理。

推荐回复格式：

```text
当前模块：M0 项目骨架

1. [已完成] 初始化 Git 仓库。
2. [当前任务] 创建 `backend/`，初始化 FastAPI 项目。

完成后请把执行结果发给我，我再给你下一项。
```

## 例外

除已经长期授权的前端工作和本地运行默认授权外，只有当用户在当前对话中明确要求 AI 助手代为执行某项具体操作时，AI 助手才可以执行该操作。额外授权仅适用于当次明确指定的范围，不视为长期授权。
